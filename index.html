نظرة عامة

هذه صفحة وثائق أحادية الملف مناسبة للنشر مباشرة على GitHub Pages. الواجهة مصممة RTL بشكل افتراضي مع زر تبديل للإنجليزية. لا تحتاج إلى أصول خارجية — الشعار مضمّن داخل الملف.

مزايا

  • صفحة ثابتة، سهلة النشر على GitHub Pages.
  • RTL افتراضي مع دعم تبديل اللغة إلى LTR/EN.
  • ألوان العلامة التجارية مطبقة: #0A47A3 و #09A66D.

كيفية النشر

  1. أنشئ repo باسم مثل digital-pins.github.io أو فعّل GitHub Pages على أي repo.
  2. ضع هذا الملف كـ index.html على الفرع الذي تخدمه Pages (عادة main أو gh-pages).
  3. انتظر بضع دقائق ثم زر عنوان Pages.

اختبار محلي سريع

python3 -m http.server 8000
# ثم فتح http://localhost:8000
